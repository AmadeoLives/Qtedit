# Qtedit
Qtedit (pronounced as "cute edit") is a W.I.P., simple, minimalistic text editor made in QtWidgets, similar to GNOME's gedit.

# Build instructions
Linux:
```shell
mkdir build
cd build
make -j$(nproc)
```
